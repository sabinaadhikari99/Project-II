# SkillSync AI

SkillSync AI is an AI-powered career development and job-matching platform designed to help users enhance their professional profiles, generate ATS-friendly CVs, discover job opportunities, and connect with employers. The platform streamlines the recruitment process by combining intelligent profile analysis, CV generation, skill management, and job recommendations in a single application.

# Setup 
Clone Repository
```bash
git clone https://github.com/sabinaadhikari99/Project-II.git
cd Project-II
```

# Setup Instructions
```bash
uv init
uv sync 
uv run python manage.py migrate
code .
```
# Run Development Server
```bash
uv run python manage.py runserver
```

Open your browser and visit:
```text
http://127.0.0.1:8000/
```
